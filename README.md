# moyu_read
一个基于Python和PyQt6开发的本地文本阅读器

## 功能特点

### 🎯 核心功能
- **无边框设计**：简洁的阅读界面，支持置顶显示
- **多种翻页模式**：支持鼠标左键翻页或移动窗口（可配置）
- **章节导航**：自动识别小说章节，支持快速跳转
- **自定义阅读体验**：可调整字体、大小、颜色、背景透明度
- **阅读进度保存**：自动保存阅读位置，支持多文件进度管理

### ⚙️ 个性化设置
- **字体设置**：支持多种中英文字体，可调整大小
- **颜色主题**：自定义背景色和字体颜色，支持自动对比度调整
- **窗口透明度**：0.1-1.0多档透明度调节
- **鼠标操作**：左键可设置为移动窗口或翻页，右键始终可移动窗口
- **目录面板**：可显示/隐藏章节导航面板

### ⌨️ 快捷键支持
- `↑/↓`：上一页/下一页
- `Alt + ↑/↓`：增大/减小字体
- `Ctrl + S`：打开/关闭设置
- `Ctrl + M`：最小化窗口
- `Ctrl + D`：显示/隐藏目录
- `ESC`：关闭程序

## 安装与运行

### 系统要求
- Python 3.7+
- Windows/Linux/macOS

### 安装步骤

1. **安装依赖**：
pip install -r requirements.txt

## 使用说明

### 基本操作
1. **打开文件**：在配置文件中设置`file_path`指向您的文本文件
2. **移动窗口**：
   - 左键拖动（当左键模式设为"移动窗口"时）
   - 右键拖动（任何模式下都可用）
3. **调整大小**：中键拖动窗口边缘
4. **翻页阅读**：使用方向键或鼠标点击（取决于左键模式设置）

### 章节功能
- 程序会自动识别文本中的章节标题（支持多种章节格式）
- 点击目录面板中的章节标题可快速跳转
- 目录面板宽度可调整，可隐藏以节省空间

### 个性化定制
通过设置窗口可以：
- 选择系统字体或使用字体对话框选择更多字体
- 调整背景透明度获得更好的阅读体验
- 开启自动字体颜色功能，根据背景色自动优化对比度
- 设置鼠标左键的默认行为（移动或翻页）

## 技术实现

### 核心架构
- **GUI框架**：PyQt6提供跨平台界面支持
- **文本处理**：使用chardet自动检测文件编码
- **热键管理**：keyboard库实现全局快捷键
- **配置管理**：JSON格式的持久化配置存储

### 特色功能实现
- **智能章节识别**：基于正则表达式匹配多种章节标题格式
- **自适应颜色**：根据背景亮度自动选择最佳字体颜色
- **无边框窗口**：自定义标题栏按钮，支持移动和调整大小
- **多文件进度管理**：为每个文件独立保存阅读位置

## 注意事项

1. **首次使用**：请确保`config.json`中的`file_path`指向有效的文本文件
2. **编码支持**：程序自动检测文件编码，支持GBK、UTF-8等常见编码
3. **性能优化**：大型文件建议开启目录功能以便快速导航
4. **全局热键**：部分系统可能需要权限授权才能使用全局快捷键
